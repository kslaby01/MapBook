    • Getting Started

    Welcome to MapBook. To get started, select a hexagon from the Key menu. The larger of the two hexagons is the current painting color. If none are large, then the default 'grey' will be painted instead.

    To change a color select the box and highlight the color in the color wheel. When done, press out of the box and the color will be applied.
